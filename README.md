# Trello frontend
Created using `create-react-app`

## Pre-requisites 

* nodejs v14.15.0 (Used for development)
  
  
## Steps to setup locally

### Run the app
* `git clone https://github.com/jeeeevs/trello_frontend.git`
* `cd trello_frontend`
* `npm i`
* `npm start`
* Site will be available `http://localhost:3001`
